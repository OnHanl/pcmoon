Загрузить CSV с ценами комплектующих

Скачайте текущий каталог кнопкой «Скачать текущий каталог CSV», отредактируйте цены и наличие в Excel/Google Таблицах (столбцы price и in_stock), затем загрузите файл обратно.

Строки с указанным id обновят существующие детали. Строки без id (или с пустым значением) создадут новые — не забудьте правильно указать category_key (ключ категории — посмотреть можно в разделе «Категории комплектующих»).
